# Thesis Title: [Eksplorasi Pencarian Partikel Baru di LHC]

## Description
Disertasi ini melakukan studi pencarian partikel baru di LHC, dengan menggunakan data eksperimen yang telah dilakukan oleh ATLAS dan CMS. Disini akan dilampirkan beberapa contoh kode yang digunakan dalam perhitungan terkait dengan parameter-parameter dalam disertasi, baik untuk pencarian $\Z'$ maupun DNN-assisted $H \rightarrow \mu \tau$.
